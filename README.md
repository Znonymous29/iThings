# 物联网云平台go-things

#### 介绍


依照腾讯云物联网平台打造,无缝迁移,基于go-zero框架,目前支持mqtt协议,支持数据模板检验,日志记录,数据流转,实时数据反馈,低代码数据流转,用户及设备互联互通.使用了kafka,redis,mysql,MongoDB,etcd开源框架.
git地址:[https://gitee.com/godLei6/things](https://gitee.com/godLei6/things)
